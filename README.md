# JackkillianAutoPackager
### Make Pip packages with ease!
  
[On Github](https://github.com/Jackkillian/Jackkillian-Auto-Packager)  
Install with pip: ```pip install JackkillianAutoPackager```  
To run:  
```import AutoPackager```  
```AutoPackager.main()```
